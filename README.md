qwerty is an open source tool to vibe-manage VPS for computes, backends serving as ai agents and more.

## Use Cases

- **Backends & APIs**: Host Nginx, FastAPI, Django, or Node.js applications with PM2/systemd process management.
- **Compute & AI Nodes**: Run ML inference engines (Ollama, vLLM), database servers, or custom script workers.
- **AI Agents & Bots**: Run persistent autonomous agents, background scraping scripts, or Telegram/Discord bots.
- **Development & CI/CD**: Deploy staging environments, remote development boxes, or self-hosted pipeline runners.

## Features

- Connect to any VPS via SSH.
- Keep session logs and AI memory.
- Conversational REPL using Anthropic Claude.
- Built-in profiles for AWS, Hetzner, DO, etc.
- Slash commands like `/mode`, `/diff`, `/memory`, etc.
